## 快捷键

<kbd>Command/Control</kbd> <kbd>Shift</kbd> <kbd>O</kbd> : 全局快捷键，显示/隐藏电子微博<br>
<kbd>Shift</kbd> <kbd>W</kbd> : 发布新微博

## Build

```bash
$ cd electronic-weibo
$ npm install

# run app in development mode
$ npm run app

# Build for Windows
$ npm run build:win

# Build for OSX
$ npm run build:osx
```
